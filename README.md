# 🍽️ Swiggy Sales Dashboard (Excel)

## 📌 Project Overview
This project is an interactive **Swiggy Sales Dashboard** built in **Microsoft Excel** to analyze restaurant sales, customer ratings, and order trends. The dashboard provides valuable business insights using Pivot Tables, Pivot Charts, KPIs, and Slicers, enabling users to explore sales performance across different states, cities, categories, and time periods.

---

## 🎯 Objectives
- Analyze overall sales performance.
- Monitor customer ratings and rating counts.
- Track monthly sales trends.
- Compare sales across states and cities.
- Identify top-performing food categories and restaurants.
- Build an interactive dashboard for quick business insights.

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

---

## 📊 Dashboard KPIs

The dashboard displays the following key metrics:

- 💰 **Total Sales**
- ⭐ **Average Rating**
- 📝 **Total Rating Count**
- 📦 **Total Orders**
- 💵 **Average Order Value (AOV)**

---

## 📈 Dashboard Features

- Interactive slicers for dynamic filtering
- Monthly sales trend analysis
- State-wise sales analysis
- City-wise sales analysis
- Restaurant performance comparison
- Food category analysis
- Customer rating insights
- Clean and easy-to-understand dashboard layout

---

## 📂 Dataset Information

The dataset contains order-level information including:

- State
- City
- Order Date
- Day
- Week
- Quarter
- Restaurant Name
- Restaurant Location
- Food Category
- Dish Name
- Food Type (Veg/Non-Veg)
- Price (INR)
- Rating
- Rating Count

## 📌 Business Insights

The dashboard helps answer questions such as:

- Which month generated the highest sales?
- Which states contribute the most revenue?
- Which cities have the highest number of orders?
- Which restaurants perform the best?
- Which food categories are most popular?
- What is the average customer rating?
- How does Average Order Value change across filters?
